# Smart Nutrition

Smart Nutrition is an AI-powered calorie tracking and nutrition management application.

## Project Overview

Smart Nutrition helps users track their daily calorie intake, analyze meal nutrition, and provide personalized health insights.

## Features

- Calorie tracking
- Meal logging
- Nutrition analysis
- AI-powered meal suggestions

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

```sh
# Clone the repository
git clone https://github.com/Hotwodi/AICalorieTracker.git

# Navigate to the project directory
cd smart-nutrition

# Install dependencies
npm install

# Start the development server
npm run dev
```
